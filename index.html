Reflexão e Aplicação Prática

Propriedades CSS utilizadas

  • display: flex; – Define um contêiner flexível.
  • flex-direction – Define a direção dos itens (linha ou coluna).
  • justify-content – Alinha os itens no eixo principal.
  • align-items – Alinha os itens no eixo transversal.
  • flex-wrap – Controla se os itens devem quebrar linha.
  • align-self – Alinha um item individual de forma diferente dos outros.

Aplicação prática com Flexbox

Se eu fosse criar uma galeria de fotos responsiva, usaria Flexbox para alinhar as imagens lado a lado em telas grandes, e empilhá-las em telas pequenas. Usaria `display: flex` no contêiner da galeria, `flex-wrap: wrap` para permitir a quebra de linha e `justify-content: space-between` para distribuir o espaço entre elas. Assim, o layout se ajustaria automaticamente ao tamanho da tela.
